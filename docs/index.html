Namespace ASEAssignment

Classes

AppArray

This is a custom implementation of BOOSE.Array class with updated restrictions limit. This class ensures that no more than 1000 arrays can be created.

AppCall

AppCall is a custom implementation of the BOOSE.Call command with no additional restrictions.

AppCanvas

The AppCanvas class implements the ICanvas interface for drawing shapes and managing pen position a graphical canvas. It provides functionalities for drawing lines, circles and other geometrical shapes. Implemented using singleton pattern to maintain a single canvas state throughout the application.

AppCircle

Represents a command for drawing a circle with optional fill on the canvas. This class inherits the BOOSE.CommandOneParameter class of BOOSE.

AppCommandFactory

A custom factory for creating commands in the BOOSE language. This factory extends the base BOOSE.CommandFactory and overrides the MakeCommand(string) method to add custom command creation logic, such as "write" command for now.

AppElse

AppElse class is a custom implementation of the BOOSE.Else command with added restrictions.

AppEnd

AppEnd class is a custom implementation of the BOOSE.End command with no additional restrictions.

AppFor

Custom implementation of BOOSE BOOSE.For class.

AppIf

Custom implementation of the BOOSE.If class. This class helps in reducing the restrictions imposed on the original If class.

AppInt

Represents an extended implementation of the BOOSE.Int class in the BOOSE framework. This class adds a restriction on the number of instances that can be created and handles compilation and execution.

AppMethod

AppMethod a custom implementation of the BOOSE.Method class with a limit on the number of methods allowed.

AppReal

Represents a custom implementation of the BOOSE.Real class with modified restrictions.

AppRect

Represents a command to draw a rectangle on the canvas, either filled or unfilled.

AppStoredProgram

Custom implementation of the BOOSE.StoredProgram class. This child class is created in order to redefine the execution limit of the loop.

AppTriangle

Draw a triangle of provided width and heigh with the current cursor position being the top vertex. The pen position will not change

AppWhile

Custom implementation of the BOOSE.While class.

AppWrite

AppWrite class a custom command for writing text on the canvas or the output window. It inherits the CommandOneParameter class and overrides the parent methods.

Form1

Represents the main form of the application. Provides functionality for executing and visualizing a BOOSE program on a canvas.
